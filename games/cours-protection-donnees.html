🔒 Protection des données

Comprendre quelles informations partager, et comment protéger vos données sur internet.

⏱️ 08:30 - 10:00
📍 Salle 6
👨‍🏫 Alexis Höppeler

📚 Introduction

Quand vous cherchez un emploi, vous donnez des informations personnelles : nom, adresse, téléphone, CV, permis, diplômes.

Ces informations sont des données personnelles. Il faut les protéger.

Donnée personnelle = information qui parle de vous.

Objectif : partager ce qui est nécessaire, au bon endroit, et éviter les arnaques.

🔤 Mots importants

Voici des mots utiles pour la protection des données :

Données personnelles

Nom, téléphone, adresse, e-mail, date de naissance, CV.

Confidentialité

Vos informations ne doivent pas être partagées avec tout le monde.

Mot de passe

Une phrase secrète pour protéger un compte.

Double vérification (2FA)

Un code en plus (SMS ou application) pour se connecter.

Phishing

Faux e-mail/SMS qui veut voler vos mots de passe.

Pièce jointe

Un fichier envoyé : CV, diplôme, permis.

HTTPS

Le cadenas 🔒 dans le navigateur : connexion protégée.

Arnaque

Un site ou une personne qui veut vous tromper.

Règle simple :
Si vous avez un doute, ne donnez pas l’information et demandez de l’aide.

✅ Bonnes pratiques

Avant de commencer

  • Préparez vos documents : CV, diplômes, permis
  • Utilisez un mot de passe long (phrase) pour votre e-mail
  • Activez la double vérification (2FA) si possible

Quand vous postulez en ligne

  • Vérifiez le site : adresse correcte + cadenas 🔒 (HTTPS)
  • Ne donnez jamais votre mot de passe à quelqu’un
  • Envoyez seulement les documents nécessaires
  • Évitez le Wi‑Fi public pour envoyer des documents importants
  • Gardez une copie de ce que vous envoyez
Important : un employeur sérieux ne demande pas votre mot de passe, ni un paiement.

🕵️‍♀️ Reconnaître une arnaque

Les arnaques (phishing) peuvent arriver par e-mail, SMS ou sur un site.

Signaux d’alerte

  • Message très urgent : “Dernière chance !”, “Votre compte va être fermé”
  • On vous demande un mot de passe ou un code
  • Adresse étrange (beaucoup de lettres/chiffres) ou faute dans le nom du site
  • Pièce jointe inattendue (surtout .exe, .zip)
  • On vous demande de payer pour postuler
Conseil : ne cliquez pas. Ouvrez le site vous‑même (en tapant l’adresse) ou demandez à votre conseiller ORP.

🧾 Que partager (et quoi éviter)

Souvent OK pour postuler

  • Nom, prénom
  • Téléphone et e-mail
  • CV (expérience, formation)
  • Permis/diplômes si demandé

À éviter (sauf si vraiment nécessaire)

  • Numéro de carte bancaire
  • Mots de passe ou codes
  • Photos de documents envoyées à n’importe qui
  • Numéros sensibles (ex. AVS) si ce n’est pas demandé officiellement
Astuce : si vous envoyez un document, donnez le bon fichier (pas toute votre galerie photo).

📌 À retenir

  • Vos données personnelles doivent être protégées
  • Ne donnez jamais vos mots de passe ou vos codes
  • Vérifiez l’adresse du site et le cadenas 🔒 (HTTPS)
  • En cas de doute : ne cliquez pas et demandez de l’aide
  • Envoyez seulement les documents nécessaires
  • Gardez une copie de ce que vous envoyez
