📚 Introduction

Sur un formulaire en ligne (jobs.ch, jobup.ch, site d’entreprise), vous devez souvent joindre un fichier.

Le plus souvent : CV (PDF) et parfois diplômes ou permis.

Joindre un fichier = ajouter un document à votre formulaire avant d’envoyer.

🖥️ Joindre un fichier – étape par étape

1
Préparez le bon fichier

Gardez votre CV en PDF avec un nom simple : CV-Nom-Prénom.pdf.

2
Trouvez le bouton

Cherchez : Joindre, Ajouter, Télécharger, Upload, Parcourir…

3
Choisissez le fichier

Une fenêtre s’ouvre : allez dans Téléchargements ou Documents, puis sélectionnez votre CV (PDF).

4
Vérifiez que le fichier est ajouté

Le site affiche souvent le nom du fichier : CV-Nom-Prénom.pdf. Si vous ne le voyez pas, ce n’est pas ajouté.

5
Si ça ne marche pas

Vérifiez le format (PDF), la taille (MB) et renommez le fichier si besoin (pas d’accents, pas de caractères spéciaux).

6
Envoyez

Quand tout est OK, cliquez sur Envoyer / Soumettre, puis gardez une preuve (capture d’écran).

Conseil : envoyez seulement les documents demandés. En cas de doute, demandez à votre conseiller ORP.

🌐 Où trouver les formulaires ?

  • jobs.ch – Cliquez sur une offre, puis "Postuler" ou "Bewerben"
  • jobup.ch – Cliquez sur "Candidater" dans l'offre d'emploi
  • rav.ch – Pour les formulaires ORP officiels
  • Site de l'entreprise – Cherchez "Emplois" ou "Carrières" ou "Jobs"
  • À l'ORP – Formulaires papier ou sur l'ordinateur de l'ORP
Aide : Votre conseiller ORP peut vous montrer comment trouver et remplir ces formulaires sur l'ordinateur.

⚠️ Erreurs fréquentes

Laisser une case * vide. Le formulaire n'est pas envoyé. Vérifiez toutes les cases obligatoires avant d'envoyer.
Écrire une mauvaise adresse e-mail. L'employeur ne peut pas vous répondre. Vérifiez votre e-mail lettre par lettre.
Oublier de joindre le CV. Vérifiez que votre CV est bien ajouté avant d'envoyer.
Mauvais format (ex: photo, fichier .zip). Utilisez PDF pour le CV si possible.
Fichier trop lourd. Si le site refuse, demandez de l’aide pour réduire la taille (PDF plus léger).
Fermer la page avant d'envoyer. Vous perdez tout ce que vous avez écrit. Cliquez d'abord sur "Envoyer".

✍️ Exercice pratique

Joindre un CV sur un formulaire

Situation : Vous avez trouvé un emploi sur jobs.ch. Le formulaire demande : “Joindre CV (PDF)”.

Tâches :

  1. Créez/choisissez un fichier : CV-Nom-Prénom.pdf
  2. Allez sur jobs.ch et ouvrez une offre
  3. Cliquez sur “Postuler”
  4. Trouvez “Joindre” / “Télécharger”
  5. Sélectionnez le PDF sur l’ordinateur
  6. Vérifiez que le nom du fichier apparaît
  7. Montrez à votre conseiller ORP avant d’envoyer

📌 À retenir

  • Préparez vos documents avant de commencer
  • Lisez tout le formulaire avant d'écrire
  • * = obligatoire, vous devez remplir cette case
  • Vérifiez votre adresse e-mail et votre numéro de téléphone
  • Joignez votre CV en PDF
  • Gardez une copie du formulaire envoyé
  • Si vous ne comprenez pas, demandez à votre conseiller ORP
