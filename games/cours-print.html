📚 Introduction

Quand vous cherchez un emploi, vous devez souvent remplir des formulaires. Par exemple sur jobs.ch, sur le site d'une entreprise, ou à l'ORP.

Ce cours vous montre comment remplir ces formulaires. Étape par étape, c'est simple !

Formulaire de candidature = une liste de questions sur vous. L'employeur veut connaître votre nom, votre expérience, vos compétences.

🔤 Mots que vous voyez dans un formulaire

Nom de famille
Votre nom
Exemple : DUPONT (en majuscules)
Prénom
Votre premier nom
Exemple : Marie
Date de naissance
Le jour où vous êtes né(e)
Exemple : 15.03.1985
Nationalité
Votre pays d'origine
Exemple : Marocaine
Permis de travail
Votre type de permis en Suisse
Exemple : Permis B, Permis C, Permis F
Taux d'activité
Combien d'heures vous voulez travailler
Exemple : 100% (temps complet) ou 50% (mi-temps)
Disponibilité
Quand vous pouvez commencer
Exemple : Immédiatement / 01.04.2025
* obligatoire
Cette case doit être remplie
Si vous ne remplissez pas, le formulaire n'est pas envoyé

🖥️ Remplir un formulaire en ligne – étape par étape

1Préparez vos documents

Avant de commencer, préparez : votre CV, votre permis de séjour, votre numéro AVS, vos diplômes.

2Lisez tout le formulaire

Lisez toutes les questions avant d'écrire. Ainsi vous comprenez ce qui est demandé.

3Remplissez les cases

Cliquez dans la case blanche. Écrivez vos informations. Utilisez la touche Tab pour aller à la case suivante.

4Joignez vos documents

Cherchez le bouton "Joindre" ou "Télécharger". Ajoutez votre CV en PDF. Parfois aussi votre diplôme.

5Vérifiez tout

Lisez votre formulaire une fois. Vérifiez que toutes les cases * sont remplies. Vérifiez votre e-mail et numéro de téléphone.

6Envoyez

Cliquez sur "Envoyer" ou "Soumettre". Attendez le message de confirmation. Gardez une copie (capture d'écran ou impression).

Conseil : Si vous ne comprenez pas une question, laissez la case vide et demandez à votre conseiller ORP. Ne devinez pas.

🌐 Où trouver les formulaires ?

  • jobs.ch – Cliquez sur une offre, puis "Postuler" ou "Bewerben"
  • jobup.ch – Cliquez sur "Candidater" dans l'offre d'emploi
  • rav.ch – Pour les formulaires ORP officiels
  • Site de l'entreprise – Cherchez "Emplois" ou "Carrières" ou "Jobs"
  • À l'ORP – Formulaires papier ou sur l'ordinateur de l'ORP
Aide : Votre conseiller ORP peut vous montrer comment trouver et remplir ces formulaires sur l'ordinateur.

⚠️ Erreurs fréquentes

Laisser une case * vide. Le formulaire n'est pas envoyé. Vérifiez toutes les cases obligatoires avant d'envoyer.
Écrire une mauvaise adresse e-mail. L'employeur ne peut pas vous répondre. Vérifiez votre e-mail lettre par lettre.
Oublier de joindre le CV. Vérifiez que votre CV est bien ajouté avant d'envoyer.
Fermer la page avant d'envoyer. Vous perdez tout ce que vous avez écrit. Cliquez d'abord sur "Envoyer".

📌 À retenir

  • Préparez vos documents avant de commencer
  • Lisez tout le formulaire avant d'écrire
  • * = obligatoire, vous devez remplir cette case
  • Vérifiez votre adresse e-mail et votre numéro de téléphone
  • Joignez votre CV en PDF
  • Gardez une copie du formulaire envoyé
  • Si vous ne comprenez pas, demandez à votre conseiller ORP

📝 Remplir un formulaire

⏱️ 08:30 - 10:00
📍 Salle 6
👨‍🏫 Alexis Höppeler

📚 Introduction

En Suisse, vous devez souvent remplir des formulaires. Par exemple : un formulaire pour l'ORP, pour un emploi, ou pour une école.

Un formulaire, c'est une liste de questions. Vous écrivez vos réponses dans des cases.

Formulaire = une feuille avec des questions à remplir. Vous écrivez vos informations dans les cases vides.

🔤 Mots importants

Voici les mots que vous voyez souvent dans un formulaire :

Nom de famille

Votre nom. Exemple : Dupont

Prénom

Votre premier nom. Exemple : Marie

Date de naissance

Le jour où vous êtes né(e). Exemple : 15.03.1985

Adresse

Où vous habitez. Exemple : Rue de la Paix 5, 1200 Genève

Numéro de téléphone

Votre numéro. Exemple : 079 123 45 67

Adresse e-mail

Votre e-mail. Exemple : marie.dupont@gmail.com

Nationalité

Votre pays d'origine. Exemple : Marocaine

Signature

Votre nom écrit à la main, à la fin du formulaire

* (étoile) = obligatoire. Vous devez remplir cette case. Si vous ne remplissez pas, le formulaire n'est pas accepté.

✏️ Comment remplir un formulaire – étape par étape

Avant de commencer :

  • Lisez tout le formulaire une fois
  • Préparez vos documents : carte d'identité, permis de séjour, AVS
  • Prenez un stylo bleu ou noir (pas de crayon)

Pour remplir :

  • Écrivez en lettres majuscules si demandé : DUPONT MARIE
  • Écrivez la date comme en Suisse : jour.mois.année → 15.03.1985
  • Cochez la bonne case avec ✓ ou X
  • Ne laissez pas de cases vides avec * (obligatoire)
  • Écrivez votre signature à la fin
Si vous ne comprenez pas une question : laissez la case vide et demandez à votre conseiller ORP. Ne devinez pas.

🏢 Formulaires à l'ORP

À l'ORP, vous avez peut-être déjà rempli des formulaires. Voici les plus courants :

  • Formulaire d'inscription : votre nom, adresse, situation de famille, formation
  • Rapport de recherche d'emploi : les emplois que vous avez cherchés ce mois-ci
  • Formulaire de candidature : pour postuler à un emploi

Le rapport de recherche d'emploi :

Chaque mois, vous écrivez les emplois que vous avez cherchés. Vous écrivez :

  • Le nom de l'entreprise
  • La date de votre candidature
  • Comment vous avez postulé (e-mail, site, courrier)
  • La réponse que vous avez reçue
Important : Gardez une copie de toutes vos candidatures. Votre conseiller ORP va vérifier.

💻 Formulaires sur internet

Beaucoup de formulaires sont sur internet. Par exemple sur jobs.ch ou rav.ch.

  • Cliquez dans la case blanche pour écrire
  • Utilisez la touche Tab pour aller à la case suivante
  • Les cases avec * sont obligatoires
  • Cliquez sur "Envoyer" ou "Soumettre" à la fin
  • Attendez le message de confirmation
Conseil : Faites une capture d'écran ou imprimez la confirmation. C'est votre preuve que vous avez envoyé le formulaire.

📌 À retenir

  • Formulaire = feuille avec des cases à remplir
  • * étoile = obligatoire, vous devez remplir
  • Écrivez la date : jour.mois.année → 15.03.1985
  • En Suisse : majuscules si demandé
  • Signez à la fin du formulaire
  • Gardez une copie de chaque formulaire envoyé
  • Si vous ne comprenez pas, demandez à votre conseiller ORP

Pour chercher un emploi, vous avez besoin de mots importants.

Ce cours vous apprend les mots de base pour comprendre une offre d'emploi et remplir un formulaire.

Niveau A1 : Ces mots sont simples. Lisez chaque mot. Lisez l'explication. Lisez l'exemple.

💼 Mots importants pour chercher un emploi

MotExplication simpleExemple
Emploi / TravailUne activité pour gagner de l'argent"Je cherche un emploi de cuisinier"
EmployeurLa personne ou l'entreprise qui donne le travail"Mon employeur s'appelle Restaurant Michel"
CandidatureUne demande pour avoir un emploi"J'envoie ma candidature par e-mail"

🏢 Mots utiles dans une offre d'emploi

Mots sur le type de travail

  • Temps complet / 100% : Travail 5 jours par semaine, 8h par jour
  • Temps partiel / 50% ou 80% : Travail moins de 5 jours par semaine
  • CDI : Contrat sans limite de temps (contrat permanent)
  • CDD : Contrat avec une date de fin (contrat temporaire)
  • Intérim : Travail temporaire par une agence

Mots sur les qualités demandées

  • Ponctuel(le) : Toujours à l'heure
  • Motivé(e) : Avec envie de travailler
  • Autonome : Peut travailler seul(e) sans aide
  • Dynamique : Actif(ve) et rapide
  • Flexible : Peut changer ses horaires si nécessaire

Mots sur les langues

  • Langue maternelle : La langue de votre pays (exemple : arabe, portugais, tamoul)
  • Courant : Vous parlez très bien
  • Notions / Débutant : Vous comprenez un peu
  • A1, A2, B1, B2... : Le niveau de la langue (A1 = débutant, B2 = bien)

📌 À retenir

  • Candidature = demande pour un emploi
  • CV = document avec votre expérience
  • Formulaire = cases à remplir
  • Entretien = réunion avec l'employeur
  • Commencez un e-mail avec "Bonjour Madame / Monsieur"
  • Finissez un e-mail avec "Cordialement"
  • Si vous ne comprenez pas, demandez : "Pouvez-vous répéter ?"
