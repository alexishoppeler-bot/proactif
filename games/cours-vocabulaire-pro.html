📚 Introduction

Pour chercher un emploi, vous avez besoin de mots importants.

Ce cours vous apprend les mots de base pour comprendre une offre d'emploi et remplir un formulaire.

Niveau A1 : Ces mots sont simples. Lisez chaque mot. Lisez l'explication. Lisez l'exemple.

💼 Mots importants pour chercher un emploi

Mot Explication simple Exemple
Emploi / Travail Une activité pour gagner de l'argent "Je cherche un emploi de cuisinier"
Employeur La personne ou l'entreprise qui donne le travail "Mon employeur s'appelle Restaurant Michel"
Candidature Une demande pour avoir un emploi "J'envoie ma candidature par e-mail"
CV Un document avec votre expérience et vos compétences "Mon CV montre mes anciens emplois"
Formulaire Un document avec des cases à remplir "Je remplis le formulaire sur Internet"
Expérience Les emplois que vous avez déjà faits "J'ai 3 ans d'expérience en cuisine"
Compétence Ce que vous savez faire "Je parle français et anglais"
Entretien Une réunion avec l'employeur pour parler du travail "J'ai un entretien lundi à 10h"

🏢 Mots utiles dans une offre d'emploi

Mots sur le type de travail

  • Temps complet / 100% : Travail 5 jours par semaine, 8h par jour
  • Temps partiel / 50% ou 80% : Travail moins de 5 jours par semaine
  • CDI : Contrat sans limite de temps (contrat permanent)
  • CDD : Contrat avec une date de fin (contrat temporaire)
  • Intérim : Travail temporaire par une agence

Mots sur les qualités demandées

  • Ponctuel(le) : Toujours à l'heure
  • Motivé(e) : Avec envie de travailler
  • Autonome : Peut travailler seul(e) sans aide
  • Dynamique : Actif(ve) et rapide
  • Flexibles : Peut changer ses horaires si nécessaire

Mots sur les langues

  • Langue maternelle : La langue de votre pays (exemple : arabe, portugais, tamoul)
  • Courant : Vous parlez très bien
  • Notions / Débutant : Vous comprenez un peu
  • A1, A2, B1, B2... : Le niveau de la langue (A1 = débutant, B2 = bien)

📌 À retenir

  • Candidature = demande pour un emploi
  • CV = document avec votre expérience
  • Formulaire = cases à remplir
  • Entretien = réunion avec l'employeur
  • Commencez un e-mail avec "Bonjour Madame / Monsieur"
  • Finissez un e-mail avec "Cordialement"
  • Si vous ne comprenez pas, demandez : "Pouvez-vous répéter ?"
